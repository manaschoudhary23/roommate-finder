# 🏠 Roommate Finder Platform

A full-stack web application that simplifies the process of finding compatible roommates and rental spaces.

## 🚀 Features

- 🔍 **Room Listings:** Add and browse rental listings with images, descriptions, and amenities
- 🌍 **Nearby Search:** Filter listings based on proximity using geolocation
- 🧠 **AI-Powered Suggestions:** HuggingFace-powered roommate & listing recommendations
- 📬 **Real-Time Chat:** Instant messaging using Socket.io and Pusher
- 📄 **Digital Agreement Form:** Allows users to generate roommate agreements
- 👤 **User Profiles:** Gender, budget, city, and amenity preferences stored and matched

## 🛠️ Tech Stack

- **Frontend:** React, JSX, CSS
- **Backend:** Node.js, Express, MongoDB, Mongoose
- **Real-Time:** Socket.io, Pusher
- **AI Integration:** HuggingFace Inference API
- **File Uploads:** Multer
- **Authentication:** JWT
